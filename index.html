Como Funciona?

A API e o Aplicativo Aurum são ferramentas para o gerênciamento dos cartões de crédito, onde o usuário poderá controlar o seu limite e transferir a cobrança de itens de uma fatura para outra conta bancária em um outro cartão de crédito.
